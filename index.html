Autonomous Quadcopter Flight Controller

A complete flight controller built from scratch using Arduino, implementing PID control algorithms for autonomous stabilization. This project demonstrates end-to-end systems engineering from hardware assembly to software optimization, achieving stable hover through real-time control at 250Hz.

Arduino C/C++ PID Control Sensor Fusion I2C PWM
  • Implemented independent PID loops for roll, pitch, and yaw stabilization

  • Real-time embedded programming at 250Hz update rate
